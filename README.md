# TrendCast

TrendCast is a Big Data analytics project focused on detecting trends in the electronics market using news, search data, reviews, and financial data.

## Keywords
We analyze electronics trends across:
- Product categories
- Companies
- Feature-based keywords

## Stack
- Python
- MongoDB
- PostgreSQL
- PySpark
- Flask (Dashboard)

## Team Modules
- Data Ingestion
- SQL & Financial
- NoSQL Storage
- Big Data Processing
- UI & Integration
